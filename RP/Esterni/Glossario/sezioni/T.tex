\lettera{T}

\parola{Task}{È un compito che viene assegnato a qualcuno nel team.}

\parola{Team}{Letteralmente squadra, indica un gruppo di persone che lavorano per lo stesso obiettivo.}

\parola{Template}{Può essere pensato come uno stampo o una struttura preorganizzata per facilitare la creazione di un qualcosa. Il gruppo \GRUPPO{} fa utilizzo di un template \LaTeX{} per mantenere l'omogeneità nella struttura della documentazione.}

\parola{Texmaker}{È un ambiente di sviluppo per il linguaggio \LaTeX.}

\parola{Throurhput}{È la capacità di trasmissione effettivamente utilizzata di un canale trasmissivo.}

\parola{Ticketing}{È il processo di associazione tra almeno un membro del team e un task.}

\parola{Tomcat}{È un application server che fornisce una piattaforma software per l'esecuzione di applicazioni Web sviluppate in linguaggio \gl{Java}.}

\parola{tvOS}{È un sistema operativo per la \gl{Apple} TV di quarta generazione.}
