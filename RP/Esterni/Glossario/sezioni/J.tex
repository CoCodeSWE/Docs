\lettera{J}

\parola{Java}{Linguaggio di programmazione orientato agli oggetti a tipizzazione statica, specificatamente progettato per essere il più possibile indipendente dalla piattaforma di esecuzione.}

\parola{Javascript}{Linguaggio di scripting orientato agli oggetti e agli eventi, comunemente utilizzato nella programmazione Web lato client per la creazione, in siti web e applicazioni web, di effetti dinamici interattivi tramite funzioni di script invocate da eventi innescati a loro volta in vari modi dall'utente sulla pagina web in uso.}

\parola{Jenkins}{Strumento open source di continuous integration, scritto in linguaggio Java.}

\parola{Jolie}{Linguaggio di programmazione innovativo basato sul paradigma dei microservizi.}

\parola{JSON}{Acronimo per JavaScript Object Notation, è un formato adatto all'interscambio di dati fra applicazioni client-server.}

\parola{JWT}{Acronimo per JSON Web Token, è uno standard per la creazione di sessioni d'accesso all'interno di determinati domini.}
